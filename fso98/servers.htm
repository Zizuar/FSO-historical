SUGGESTED IRC SERVERS

FSO uses one server currently, the Superchat IRC Network. Listed below are the suggested servers:
NewBrunswick.NJ.US.SuperChat.Org
NewOrleans.LA.US.SuperChat.Org
Houston.TX.US.SuperChat.Org
Kentville.NS.CA.SuperChat.Org
Melbourne.AU.SuperChat.Org
All Servers use port 6667

If you don't have an IRC client we suggest you get mIRC if you use Windows, or IRCle if you use MacOS.


MUSH SERVER

FSO is proud to be the co-runner of Star Trek: Betrayal.  STB is a Star Trek oriented MUSH based in an alternate timeline, where the UFP has just been liberated from Dominion rule, the Klingons are battle-crazy maniacs who have defeated every ship in their path, and the Dominion is staying in the GQ... for now

You can connect to STB by TELNETing to Pegasus.HPNC.Com port 1701 (type telnet://pegasus.hpnc.com:1701 in your web browser).  We recommend, however, you use a MU* client like SimpleMU or Pueblo if you really want to enjoy STB fully.
