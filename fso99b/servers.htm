SUGGESTED IRC SERVERS

FSO uses one server currently, the MagicStar IRC Network. Listed below are the suggested servers:
TCI.MagicStar.Net
NCI.MagicStar.Net
Uplink.MagicStar.Net
Gryphon.MagicStar.Net
Atlantis.MagicStar.Net
All Servers use port 6667

If you don't have an IRC client we suggest you get mIRC if you use Windows, or IRCle if you use MacOS.
